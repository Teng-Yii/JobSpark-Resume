# JobSpark-Resume
🔥 点燃求职火花！AI 驱动的简历优化 + 模拟面试工具，智能挖掘简历亮点、优化表达逻辑，实时模拟真实面试场景并给出针对性反馈，让求职之路少走弯路，快速斩获心仪 offer～
# 一、核心功能模块（README 核心内容）
## 🔥 简历智能优化：让你的简历自带 “闪光点”
- AI 简历扫描仪：上传简历后，自动分析内容完整性（如工作经历量化程度、技能匹配度），生成优化评分和改进建议。
- 关键词精准匹配：输入目标岗位 JD，工具自动提取高频关键词，并提示在简历中自然嵌入，提升 ATS 系统通过率。
## 🎯 模拟面试演练场：提前 “彩排”，面试不慌
- 多场景面试官：支持按岗位（产品经理 / 算法工程师 / 运营等）、公司风格（大厂压力面 / 初创公司业务面）生成 AI 面试官，模拟真实对话节奏。
- 实时反馈雷达：面试结束后，从 “表达逻辑性”“岗位匹配度”“问题回应深度” 等维度生成评分报告，附具体改进点（例：“回答‘职业规划’时可结合公司业务方向，增强说服力”）。
- 错题本功能：自动记录未答好的问题，生成针对性练习题库，支持反复模拟直至熟练。
